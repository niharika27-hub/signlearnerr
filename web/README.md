# React Scroll Landing (Frontend)

This frontend uses **React + TypeScript + Tailwind CSS + Framer Motion** to build a scroll-animated landing page.

## Run locally

- `npm install`
- `npm run dev`

## Build & lint

- `npm run build`
- `npm run lint`

## Key locations

- Shared animation component: `src/components/ui/container-scroll-animation.tsx`
- Hero section usage: `src/components/sections/hero-scroll-demo.tsx`
- Additional section usage: `src/components/sections/feature-scroll-demo.tsx`
- Landing composition: `src/App.tsx`
- Setup and integration notes: `docs/landing-implementation-notes.md`
